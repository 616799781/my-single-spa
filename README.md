# 简单的微前端框架

![demo](https://github.com/yataoZhang/my-single-spa/raw/master/demo.gif)

## 前提
```shell
$ npm i
```

## 快速开始
```shell
$ npm run watch
```
然后在浏览器中访问： http://localhost:10001/toutrial/quick/index.html

当前已支持：
1. 简单的快速上手
2. systemjs
3. Vuejs

## 开发模式
```shell
$ npm run build:dev
```

## 待续
